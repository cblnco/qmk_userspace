# OLKB-48 Planck Rev. 6 from Massdrop

This is my personal Keymap for the Planck Rev. 6

You can compile this keymap locally with QMK by running the following command:

```sh
qmk compile -kb planck/rev6_drop -km cblnco
```

Check [qmk.fm](https://qmk.fm/) to know more about this firmware.
